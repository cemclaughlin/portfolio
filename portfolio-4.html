Ambient Edge Kinect Service

Description

  • December 14, 2020

This project was designing the Kinect Service and a Unity Client for the Ambient Edge framework. I started by integrating the Kinect functionality into a program that connects to Edge. This also required converting all the original Kinect bodytracking API into Protobuf messaging formats.

It took a while but the demo shows that we got it working properly! The scene is dynamically changing according to my hand movements.

Technology

  • Azure Kinect
  • .NET
  • Google Protobuf
  • NATS Streaming
